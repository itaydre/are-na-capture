# div to Are.na

A Chrome extension that lets you capture any HTML element from a webpage and save it directly to your Are.na channels.

## Install

1. [Download the ZIP](https://github.com/itaydre/are-na-capture/archive/refs/heads/main.zip) and unzip it
2. Open Chrome and go to `chrome://extensions/`
3. Enable **Developer mode** (toggle in the top right)
4. Click **Load unpacked**
5. Select the unzipped `are-na-capture-main` folder

## Usage

1. Click the extension icon in the toolbar and **Login to Are.na**
2. Authorize the app — you're all set
3. Start capture mode:
   - Click the extension icon → **Start Capture**
   - Right-click on any page → **Capture Element to Are.na**
   - Keyboard shortcut: `Cmd+Shift+S` (Mac) / `Ctrl+Shift+S` (Windows/Linux)
4. Hover over elements to highlight them, click to capture
5. Select a channel, optionally name the block, and click **Connect**

Press `ESC` to cancel capture mode at any time.
